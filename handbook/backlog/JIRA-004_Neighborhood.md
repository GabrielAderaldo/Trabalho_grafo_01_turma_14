# JIRA-004: Mapeamento de Vizinhança

## Descrição
Implementar a extração da lista de adjacência (vizinhança) de vértices específicos e formatar para o relatório.

## Requisitos
- Criar uma função que recebe um vértice `v` e retorna uma representação (ex: String ou List) de todos os seus vizinhos `w`.
- Demonstrar o funcionamento para os primeiros 5 vizinhos do vértice 0.

## Definição de Pronto (DoD)
- O código deve passar no teste `src/tests/NeighborhoodTest.java`.
